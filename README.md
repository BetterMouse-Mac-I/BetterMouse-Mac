## Overview

BetterMouse Professional Suite is a complete mouse enhancement platform with cursor customization, per-app profiles, and advanced scroll control for power users.

Per-application profiles switch button mappings, scroll behavior, and cursor speed automatically based on the active application, optimizing mouse behavior for each workflow context.

Cursor customization adjusts size, color, and acceleration curves beyond what system settings allow, including separate horizontal and vertical sensitivity.

Scroll direction independence sets different scroll directions for mouse and trackpad, solving the common frustration of shared natural/traditional scroll settings.

BetterMouse Professional Suite is the mouse enhancement platform for power users who need comprehensive mouse customization with the per-app profiles and scroll control that productive multi-device input requires.

## Performance & Integration

BetterMouse is engineered around macOS technologies that handle performance-critical work efficiently. Tasks run in the background without slowing down the rest of your system — you can keep working while BetterMouse processes operations quietly behind the scenes.

The app integrates deeply with the macOS ecosystem out of the box:

- Launch workflows directly through **Spotlight**
- Preview files instantly with **Quick Look**
- Trigger actions from any app via the **Services menu**
- Fully compatible with **Shortcuts** for automated workflows
